# 1 BIT FELLA
1 BIT FELLA is an arcade platformer where you play as a 1bit fella dodging and weaving throuh floating platforms to survive for as long as possible! Can you beat the latest high score?

# CONTROLS
Arrow Keys -> Movement
Space -> Jump
